CSV & Excel Data Validator

Aplicación desarrollada con Angular para cargar, validar y analizar archivos CSV y XLSX.

Funcionalidades
- Carga de archivos CSV y XLSX.
- Visualización de registros en tabla.
- Validación de datos.
- Detección de campos vacíos.
- Detección de folios duplicados.
- Validación de fechas.
- Validación de montos numéricos.
- Validación de estatus permitidos.
- Resumen de información.


Tecnologías utilizadas
    Angular 21
    Angular Material
    TypeScript
    SCSS
    Papa Parse
    XLSX


Instalación
    npm install
    ng serve

La aplicación estará disponible en:

http://localhost:4200
